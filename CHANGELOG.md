## [1.0.0] - 2026-03-24

### Added
- **Project Management**: Complete project lifecycle management with 10 stages
- **Context Documents**: Store Avatar, investigation, competition, product, offer, and headline information
- **Files Management**: Hybrid system supporting local vault storage and external links
- **Progress Tracking**: Automatic percentage calculation for stages and tasks
- **Dashboard**: Global project overview with progress bars
- **Stage Management**: 10-stage process from research to scale with automatic time calculation
- **Task Management**: Task status tracking (active, missing, done, in-progress)
- **Pillar Assignment**: Assign tasks to pillars (copy, traffic, design, editing, integrations, strategist)
- **Calendar Integration**: Full calendar with month/week/day views
- **Event Scheduling**: Create and manage calendar events
- **Recurring Events**: Support for daily, weekly, and monthly recurring tasks
- **Time Boxing**: Schedule time-blocked tasks with drag-and-drop support
- **Daily Routine**: Track morning routine (prayer, English, listening skills)
- **Study Sections**: Log copy, VSL, TSL studies with notes
- **Book & Podcast Management**: Track reading progress and podcast notes
- **Todo Lists**: Manage tasks for today, this week, and later
- **Priority Levels**: High, medium, and low priority for tasks
- **Liquid Glass UI**: Beautiful purple-themed interface with glassmorphism effects
- **Responsive Design**: Works on desktop and mobile devices
- **Data Persistence**: Automatic saving to Obsidian vault
- **Settings**: Configurable default project duration and auto-save

### Technical
- TypeScript 5.9+ for type safety
- Zustand for state management with persistence
- Rollup for optimized bundling
- CSS Modules with design tokens
- Obsidian Plugin API (version 1.5.0+)

## Upgrade Guide

### From 0.1.0 to 1.0.0

No migration needed. Simply:
1. Uninstall old version
2. Install new version
3. All data is preserved in your vault
